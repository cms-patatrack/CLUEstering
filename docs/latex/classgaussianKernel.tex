\doxysubsection{Detailed Description}
Class that represents the gaussian kernel. 

The gaussian kernel is defined as\+:  \[
K_{ij} = \begin{cases}
1 & \text{if } i = j \\
\text{gaus_amplitude} \cdot \exp \left( - \frac{(dist_{ij} - \text{gaus_avg})^2}{2 \cdot \text{gaus_std}^2} \right) & \text{if } i \neq j
\end{cases}
\] where $ \text{gaus_avg} $, $ \text{gaus_std} $ and $ \text{gaus_amplitude} $ are user-\/defined parameters \begin{DoxyNote}{Note}
The user can choose the values of $ \text{gaus_avg} $, $ \text{gaus_std} $ and $ \text{gaus_amplitude} $ 
\end{DoxyNote}
