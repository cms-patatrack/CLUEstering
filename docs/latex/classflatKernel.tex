\doxysubsection{Detailed Description}
Class that represents the flat kernel. 

The flat kernel is defined as\+:  \[
 K_{ij} = \begin{cases}
 1 & \text{if } i = j \\
 \text{flat} & \text{if } i \neq j
 \end{cases}
 \] where $ \text{flat} $ is a user-\/defined parameter \begin{DoxyNote}{Note}
The user can choose the value of $ \text{flat} $ 

The flat kernel is the default kernel 
\end{DoxyNote}
