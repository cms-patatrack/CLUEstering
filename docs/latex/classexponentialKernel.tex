\doxysubsection{Detailed Description}
Class that represents the exponential kernel. 

The exponential kernel is defined as\+:  \[
K_{ij} = \begin{cases}
1 & \text{if } i = j \\
\text{exp_amplitude} \cdot \exp \left( - \text{exp_avg} \cdot dist_{ij} \right) & \text{if } i \neq j
\end{cases}
\] where $ \text{exp_avg} $ and $ \text{exp_amplitude} $ are user-\/defined parameters \begin{DoxyNote}{Note}
The user can choose the values of $ \text{exp_avg} $ and $ \text{exp_amplitude} $ 
\end{DoxyNote}
